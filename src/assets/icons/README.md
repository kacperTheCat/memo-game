# Navigation icons

Secondary nav chrome uses **Google Material Symbols Outlined** (ligatures `arrow_back`, `arrow_forward`, `close`) loaded from **`index.html`** — see [`specs/007-home-nav-briefcase-ui/research.md`](../../../specs/007-home-nav-briefcase-ui/research.md) §5 and [Google Fonts](https://fonts.google.com/icons) / ToS.

The previous **Heroicons**-based `NavBackIcon.vue` / `NavDismissIcon.vue` components were removed when nav switched to Material (feature **007** Phase 12).
